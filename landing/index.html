Kết nối Puppeteer
trong 3 dòng code

Mỗi profile khi launch sẽ expose một CDP endpoint. Kết nối trực tiếp từ Puppeteer hoặc Playwright — không cần plugin, không cần patch.

Xem docs đầy đủ → 
// 1. Start profile qua API
const { data } = await fetch(
  '/api/profiles/PROFILE_ID/start',
  { method: 'POST' }
).then(r => r.json());

// 2. Kết nối Puppeteer qua CDP
const browser = await puppeteer.connect({
  browserURL: `http://127.0.0.1:${data.cdpPort}`
});

// 3. Làm việc bình thường
const page = await browser.newPage();
await page.goto('https://facebook.com');
FAQ

Câu hỏi thường gặp

Pro5 Chrome có thực sự chống detect không?

Pro5 inject fingerprint qua Chrome extension ở document_start, override Canvas, WebGL, AudioContext, navigator properties. Hiệu quả với hầu hết các hệ thống detect thông thường. Không có phần mềm nào đảm bảo 100% — chất lượng proxy và hành vi người dùng cũng quan trọng không kém.

Hỗ trợ Chrome, Edge hay Chromium?

Hỗ trợ tất cả: Google Chrome, Microsoft Edge, CentBrowser, Chromium. Bạn có thể cấu hình nhiều runtime và chọn runtime cho từng profile.

Data profile lưu ở đâu?

Tất cả data lưu local trên máy bạn tại thư mục data/profiles/. Không có gì được gửi lên server của chúng tôi.
